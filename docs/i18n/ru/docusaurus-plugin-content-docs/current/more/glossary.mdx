# Глоссарий

В этом документе приведены определения ключевых терминов и сокращений, используемых в документации и кодовой базе Axelix.

## Термины

### Axelix
Название проекта. Также используется как обобщающий термин для обозначения всей системы в целом.

### Axelix Master
Независимое приложение, развёртываемое отдельно, которое:
- Предоставляет веб-интерфейс для Axelix
- Взаимодействует с управляемыми сервисами
- Служит центральным узлом управления
- Обеспечивает обнаружение и регистрацию сервисов
- Содержит встроенный MCP Сервер

### Axelix Spring Boot Starter
Spring Boot стартер, который включается в Spring Boot приложения для обеспечения интеграции с Axelix. Он предоставляет
необходимые возможности, которые используются Axelix Master.

### Управляемый сервис
Spring Boot приложение, которое:
- Имеет интегрированный Axelix Spring Boot Starter
- Зарегистрировано в Axelix Master и известно ему
- Может отслеживаться и управляться через Axelix
