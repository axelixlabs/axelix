# Глоссарий

В этом документе приведены определения ключевых терминов и сокращений, используемых в документации и кодовой базе Axile.

## Термины

### Axile
Название проекта. Также используется как обобщающий термин для обозначения всей системы в целом.

### Axile Master
Независимое приложение, развёртываемое отдельно, которое:
- Предоставляет веб-интерфейс для Axile
- Взаимодействует с управляемыми сервисами
- Служит центральным узлом управления
- Обеспечивает обнаружение и регистрацию сервисов

### Axile Spring Boot Starter (ASBS)
Spring Boot стартер, который включается в Spring Boot приложения для обеспечения интеграции с Axile. Он предоставляет
необходимые возможности, которые используются Axile Master.

**Псевдонимы:** ASBS, Axile SBS

### Управляемый сервис
Spring Boot приложение, которое:
- Имеет интегрированный Axile Spring Boot Starter
- Зарегистрировано в Axile Master и известно ему
- Может отслеживаться и управляться через Axile
