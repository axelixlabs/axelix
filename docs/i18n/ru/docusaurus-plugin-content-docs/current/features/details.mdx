# Детали

На странице **Детали** представлена информация об управляемом сервисе приложения Spring Boot. На этой странице в удобном
для восприятия формате отображаются технические метаданные, информация о среде выполнения и сведения о сборке.

![страница подробностей](/img/feature/details/details-main-page.png) ***Страница «Подробности» в
интерфейсе Axelix***

Страница доступна для всех пользователей, прошедших аутентификацию: `VIEWER`, `EDITOR`, `ADMIN` и `SUPER_ADMIN` могут
открыть её и запустить экспорт **Выгрузить состояние**. Обратите внимание, что файлы свойств среды и конфигурации в
экспортированном архиве по-прежнему защищены маскировкой прав доступа на уровне страницы:  вызывающие, у которых нет
`ENV_VALUES_READ` / `CONFIG_PROPS_VALUES_READ`, получают `******` вместо значений. Полную матрицу ролей и полномочий см.
в [Роли и полномочия](../setting-up-master-ui/authentication/authentication.mdx#roles-and-authorities).

## Заголовок раздела с информацией о сервисе
- **Имя сервиса**: имя сервиса. Постфикс может отличаться в зависимости от среды выполнения и среды обнаружения
Axelix Master.
- **Выгрузить состояние** Экспорт состояния сервиса в виде ZIP-архива, содержащего выбранные диагностические компоненты.
Выгрузка включает различные снимки состояния сервиса, которые можно использовать для отладки, анализа.

### Компоненты выгрузки состояния
Диалог выгрузки позволяет выбрать, какие компоненты включить в экспорт:

- **Дамп Кучи** — снимок памяти JVM Heap
- **Дамп Потоков** — текущие состояния потоков и стек-трейсы
- **Бины** — информация о Spring Beans
- **Кэши** — содержимое кэшей
- **Условия** — результаты вычисления Spring-условий
- **Свойства Конфигурации** — Spring Configuration Properties
- **Окружение** — окружение сервиса
- **Файл лога** — текущий файл лога сервиса
- **Запланированные Задачи** — информация о запланированных задачах

## Свойства

Страница работает поверх actuator-эндпоинта `axelix-details`, который добавляет Axelix Spring Boot Starter. Она доступна
через стандартные свойства Spring Boot Actuator. Полный список конечных точек Axelix и сопутствующих настроек см. в
разделе [Настройка Spring Boot
Starter](../setting-up-spring-boot-service/configuring-axelix-starter/configuring-axelix-starter.mdx):

<Tabs groupId="spring-config">
  <TabItem value="properties" label="application.properties">

```properties
management.endpoints.web.exposure.include=axelix-details
```

  </TabItem>
  <TabItem value="yaml" label="application.yaml">

```yaml
management:
  endpoints:
    web:
      exposure:
        include:
          - axelix-details
```

  </TabItem>
</Tabs>
