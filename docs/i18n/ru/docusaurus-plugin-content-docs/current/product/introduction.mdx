# Введение

Добро пожаловать в Axelix! Мы рады, что вы здесь.

Axelix — это инструмент для мониторинга и отладки в режиме выполнения для приложений на Spring Boot, написанных на Java
или Kotlin. Он подключается к вашим работающим сервисам и показывает текущее состояние каждого приложения — его бины,
источники конфигурации, транзакции, запланированные задачи, кеши, потоки и метрики — через веб-интерфейс для инженеров и
через встроенный MCP-сервер для AI-агентов. Часть этого состояния можно менять «на лету» без передеплоя.

Нам важно, как быстро вы сможете запустить Axelix. Раздел [Установка](../installation/configuring-master) проведёт вас
через запуск Master, а руководство [Spring Boot Starter](../setting-up-spring-boot-service/what-is-axelix-starter)
описывает добавление стартера в ваше приложение.

## Axelix OSS и Enterprise

Эта документация описывает как **Axelix OSS**, выпущенный под лицензией LGPL 3.0 с исходным кодом по адресу
[github.com/axelixlabs/axelix](https://github.com/axelixlabs/axelix), так и готовящуюся платную редакцию **Axelix
Enterprise** с дополнительными возможностями (в разработке — следите за новостями).
